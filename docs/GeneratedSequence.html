GeneratedSequence GeneratedSequence

A GeneratedSequence does not wrap an in-memory colllection but rather determines its elements on-the-fly during iteration according to a generator function.

Parameters

Name Type Description
generatorFn function A function which accepts an index and returns a value for the element at that position in the sequence.
length number The length of the sequence. If this argument is omitted, the sequence will go on forever.

each GeneratedSequence#each

See Sequence.each.

length GeneratedSequence#length

Returns the length of this sequence.

Returns

Type Description
number The length, or undefined if this is an indefinite sequence.
