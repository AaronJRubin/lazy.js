Lazy Lazy

Wraps an object and returns a Sequence.

  • For arrays, Lazy will create a sequence comprising the elements in the array (an ArrayLikeSequence).
  • For objects, Lazy will create a sequence of key/value pairs (an ObjectLikeSequence).
  • For strings, Lazy will create a sequence of characters (a StringLikeSequence).

Example

var fromArray = Lazy([1, 2, 4]);
// => Lazy.ArrayLikeSequence

var fromObject = Lazy({ foo: "bar" });
// => Lazy.ObjectLikeSequence

var fromString = Lazy("hello, world!");
// => Lazy.StringLikeSequence

Parameters

Name Type Description
source Array|Object|string An array, object, or string to wrap.

Returns

Type Description
Sequence The wrapped lazy object.

generate Lazy.generate

Creates a GeneratedSequence using the specified generator function and (optionally) length.

Example

var randomNumbers = Lazy.generate(Math.random);
// => sequence: (0.4838115070015192, 0.637410914292559, ...)

randomNumbers.length();
// => undefined

var countingNumbers = Lazy.generate(function(i) { return i + 1; }, 10);
// => sequence: (1, 2, ..., 10)

countingNumbers.length();
// => 10

Parameters

Name Type Description
generatorFn function The function used to generate the sequence. This function accepts an index as a parameter and should return a value for that index in the resulting sequence.
length number The length of the sequence, for sequences with a definite length.

Returns

Type Description
GeneratedSequence The generated sequence.

range Lazy.range

Creates a sequence from a given starting value, up to a specified stopping value, incrementing by a given step.

Example

var integers = Lazy.range(10);
// => sequence: (0, 1, ..., 9)

var countingNumbers = Lazy.range(1, 11);
// => sequence: (1, 2, ..., 10)

var evenNumbers = Lazy.range(2, 10, 2);
// => sequence: (2, 4, 6, 8)

Returns

Type Description
GeneratedSequence The sequence defined by the given ranges.

repeat Lazy.repeat

Creates a sequence consisting of the given value repeated a specified number of times.

Example

var hihihi = Lazy.repeat("hi", 3);
// => sequence: ("hi", "hi", "hi")

var foreverYoung = Lazy.repeat("young");
// => sequence: ("young", "young", ...)

Parameters

Name Type Description
value * The value to repeat.
count number The number of times the value should be repeated in the sequence. If this argument is omitted, the value will repeat forever.

Returns

Type Description
GeneratedSequence The sequence containing the repeated value.
