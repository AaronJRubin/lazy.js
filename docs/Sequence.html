Sequence Sequence

The Sequence object provides a unified API encapsulating the notion of zero or more consecutive elements in a collection, stream, etc.

inherit Sequence.inherit

Create a new constructor function for a type inheriting from Sequence.

Parameters

Name Type Description
ctor Function The constructor function.

Returns

Type Description
Function A constructor for a new type inheriting from Sequence.

toArray Sequence#toArray

Creates an array snapshot of a sequence.

Note that for indefinite sequences, this method may raise an exception or (worse) cause the environment to hang.

Example

var range = Lazy.range(1, 10)
// => sequence: (1, 2, ..., 9)

var array = range.toArray();
// => [1, 2, ..., 9]

Returns

Type Description
Array An array containing the current contents of the sequence.

toObject Sequence#toObject

Creates an object from a sequence of key/value pairs.

Example

var details = [["first", "Dan"], ["last", "Tao"], ["age", 29]];
var person = Lazy(details).toObject();
// => { first: "Dan", last: "Tao", age: 29 }

Returns

Type Description
Object An object with keys and values corresponding to the pairs of elements in the sequence.

forEach Sequence#forEach

Iterates over this sequence and executes a function for every element.

Example

var subordinates = [joe, bill, wendy];
Lazy(subordinates).forEach(function(s) { s.reprimand(); });

Parameters

Name Type Description
fn Function The function to call on each element in the sequence. Return false from the function to end the iteration.

map Sequence#map

Creates a new sequence whose values are calculated by passing this sequence's elements through some mapping function.

Example

var odds = [1, 3, 5];
var evens = Lazy(odds).map(function(x) { return x + 1; });
// => sequence: (2, 4, 6)

Parameters

Name Type Description
mapFn Function The mapping function used to project this sequence's elements onto a new sequence.

Returns

Type Description
Sequence The new sequence.

pluck Sequence#pluck

Creates a new sequence whose values are calculated by accessing the specified property from each element in this sequence.

Example

var people = [
  { first: "Dan", last: "Tao" },
  { first: "Bob", last: "Smith" }
];
var surnames = Lazy(people).pluck("last");
// => sequence: ("Tao", "Smith")

Parameters

Name Type Description
propertyName string The name of the property to access for every element in this sequence.

Returns

Type Description
Sequence The new sequence.

invoke Sequence#invoke

Creates a new sequence whose values are calculated by invoking the specified function on each element in this sequence.

Example

function Person(first, last) {
  this.fullName = function() {
    return first + " " + last;
  };
}

var people = [
  new Person("Dan", "Tao"),
  new Person("Bob", "Smith")
];

var fullNames = Lazy(people).invoke("fullName");
// => sequence: ("Dan Tao", "Bob Smith")

Parameters

Name Type Description
methodName string The name of the method to invoke for every element in this sequence.

Returns

Type Description
Sequence The new sequence.

reverse Sequence#reverse

Creates a new sequence with the same elements as this one, but to be iterated in the opposite order.

Note that in some (but not all) cases, the only way to create such a sequence may require iterating the entire underlying source when each is called.

Example

var alphabet = "abcdefghijklmnopqrstuvwxyz";
var alphabetBackwards = Lazy(alphabet).reverse();
// => sequence: ("z", "y", "x", ..., "a")

Returns

Type Description
Sequence The new sequence.

first Sequence#first

Creates a new sequence comprising the first N elements from this sequence, OR (if N is undefined) simply returns the first element of this sequence.

Example

function powerOfTwo(exp) {
  return Math.pow(2, exp);
}

var firstTenPowersOf2 = Lazy.generate(powerOfTwo).first(10);
// => sequence: (1, 2, 4, ..., 512)

Parameters

Name Type Description
count number The number of elements to take from this sequence. If this value exceeds the length of the sequence, the resulting sequence will be essentially the same as this one.

Parameters

Name Type Description

Returns

Type Description
Sequence The new sequence.

last Sequence#last

Creates a new sequence comprising the last N elements of this sequence, OR (if N is undefined) simply returns the last element of this sequence.

Example

var siblings = [lauren, adam, daniel, happy];
var favorite = Lazy(siblings).last();
// => happy

Parameters

Name Type Description
count number The number of items to take from the end of the sequence.

Returns

Type Description
* The new sequence (or the last element from this sequence if no count was given).

getIterator Sequence#getIterator

Creates an iterator object with two methods, moveNext -- returning true or false -- and current -- returning the current value.

This method is used when asynchronously iterating over sequences. Any type inheriting from Sequence must implement this method or it can't support asynchronous iteration.

Returns

Type Description
Iterator An iterator object.
